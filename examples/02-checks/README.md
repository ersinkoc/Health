# Health Check Examples

## database.ts

PostgreSQL database health check.

```bash
npx tsx examples/02-checks/database.ts
```

## redis.ts

Redis health check with critical flag.

```bash
npx tsx examples/02-checks/redis.ts
```

## http-api.ts

External HTTP API health check.

```bash
npx tsx examples/02-checks/http-api.ts
```

## custom.ts

Custom health checks with metadata and weighted scoring.

```bash
npx tsx examples/02-checks/custom.ts
```
